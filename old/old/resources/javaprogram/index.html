My Program

This is my grade analyzing program. I am in the process of creating it, and I will keep this site up to date with my progress.

My Program is currently coded in Java, which can only be run on any computer regardless of Operating System, as long as Java is installed. I am planning on making an IOS, Android and Windows Phone 8 Apps out of the program once I complete the Java version.

Some Features of My Program:

  • Add Multiple Terms and Courses
  • Add Multiple Assignments Easily for Each Course
  • Simple Adding and Removing of Assignments
  • Detailed Breakdown of the Effects of Every Course, Assignment, and Point on your GPA
  • Multiple GPA Scale Support
  • Multiple Term Type Support
  • Multiple Profile Support

Possible Future Features

  • Password Protection via Web-Based Login
  • Cloud Storage for Access Anywhere
  • HTML 5 Version for Access through Browser
  • Mobile Apps
  • Detailed Graphs of the Impact of Each Possible Score on an Assignment


Also if you have any issues with opening the program please download the latest version of Java from Java.com/en/download or email me at Bollesnj07@uww.edu
