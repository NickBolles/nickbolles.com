Four See

Four See is an app for students to analyze their GPA and help them to prioritize their school work.
Future: I want Four See to be the centerpiece to a college students career. In the future it will have features such as:
  • Term Scheduler, to plan out all the courses students need to take in their college career
  • Goal Assistant, to help nudge students towards their goals in the courses
  • Automatic Assignment imports, To avoid entering every assignment, which can be tiresome
  • Chat between classmates, To provide a place to ask questions about assignments
  • Improve flexibility, by adding different types of GPA Scale, Assignment Scoring etc
  • And of course, more cool stats!
Everyday I am thinking of new ideas for the future of Four See. If you have any ideas, questions, or concerns please email FourSee@NickBolles.com
I am in the process of releasing a 0.5 Beta version, which I hope to have released by the end of October 2015. From there on Four See will be continuelly updated with cool new features!

Nickbolles.com

Nickbolles.com is my personal website that includes professional and personal likes and hobbys. The main page of Nickbolles.com is just the start though. The server runs several nodeJS servers including:
  • Nickbolles.com Users Server and MongoDB Database
  • FourSee Server and MongoDB Database
  • Private Git Server
  • And more to come...
The server is an Amazon EC2 Linux instance. I have setup and documented, the instance myself and automated many of the tasks related to setup and deploying updates to the server.
